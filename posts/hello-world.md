# 첫 번째 블로그 글

안녕하세요! 이 블로그는 **마크다운 파일**을 읽어서 웹페이지로 변환하는 정적 블로그 생성기로 만들어졌습니다.

## 주요 기능

- 마크다운 파일을 HTML로 변환
- 다크 모드 지원
- 모바일 반응형 레이아웃

## 코드 예시

```javascript
const greeting = "Hello, Blog!";
console.log(greeting);
```

## 마무리

순수 HTML, CSS, JavaScript만으로 만든 블로그입니다. 프레임워크 없이도 충분히 멋진 블로그를 만들 수 있습니다!
